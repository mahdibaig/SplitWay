# Pinterest Inspiration Brief

**Purpose:** Before any mockups get made, collect 15-25 reference images so the design has a clear visual identity. This avoids the "AI slop" look that comes from designing without references.

**When to do this:** Sometime during Phase 1, before Phase 2 starts. ~30-60 minutes.

**How to use it:** Create a Pinterest board called "Splitway." Save anything that resonates. Don't filter — save everything that catches your eye. We'll filter together later when reviewing.

---

## What we're looking for

The design direction we agreed on:
- **Warm and friendly with Apple-native restraint**
- Not fintech-bro (no neon, no Robinhood-style)
- Not 2014-Splitwise (no muddy greens, dated lists)
- Calm, generous spacing, readable
- Could be sage/cream/muted accent palette, but open to other warm directions

## What to look for in each pin

When you save something, briefly note (in Pinterest's pin notes if useful) what about it caught your eye. Some lenses:

- **Color** — palette feels right
- **Typography** — heading style, big-number treatment
- **Layout** — spacing, hierarchy
- **Component style** — buttons, cards, list rows
- **Empty state** — how they handle "no data yet"
- **Charts** — visualization style
- **Personality** — small details that give it character
- **Onboarding** — how they welcome users
- **Iconography** — custom illustrations vs SF Symbols
- **Microcopy** — friendly vs sterile language

---

## What to AVOID saving (red flags)

- Generic dashboard with default iOS blue everywhere
- Crypto / trading app aesthetics (dark, neon, "to the moon")
- Splitwise / Mint screenshots (you specifically want to be different from these)
- Anything with cartoon mascots (we decided no mascots for an expense app)
- Maximum-density screens (lots of small text, crowded charts)
- Generic "AI app" look (gradient blue/purple everything)

## Deliverable

A Pinterest board with 15-25 saved images. Mark 3-5 as "favorites" that feel closest to where you want to land.

When you have the board, share it (Pinterest boards can be made public or shared via link), and we'll review together before any mockup gets made. From those references we'll pick:

1. A color palette (3 primary colors + neutrals)
2. A typography direction
3. Component style baseline (buttons, cards, list rows)
4. A few specific screen layouts to use as starting points

That's the input for Claude Design (or for me to create mockups via the visualizer).
